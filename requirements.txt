# 🌱 AgroMercados — Dashboard de Mercados Agrícolas Latinoamérica

Dashboard con **datos diarios reales** de commodities agrícolas, clima e insumos
para Centroamérica, Colombia, Venezuela y Ecuador.

---

## ⚡ Inicio rápido (3 pasos)

### Paso 1 — Obtén tu clave gratuita de Alpha Vantage

Ve a: **https://www.alphavantage.co/support/#api-key**

Solo necesitas un email y nombre. La clave llega inmediatamente.
El plan gratuito permite **25 llamadas/día** (suficiente para el dashboard con caché).

### Paso 2 — Configura tu clave

**macOS / Linux:** edita `start.sh` y reemplaza `PEGA_TU_CLAVE_AQUI` con tu clave.

**Windows:** edita `start.bat` y reemplaza `PEGA_TU_CLAVE_AQUI` con tu clave.

O pasa la clave directamente:
```bash
python server.py TU_CLAVE_AQUI
```

### Paso 3 — Ejecuta

**macOS / Linux:**
```bash
chmod +x start.sh
./start.sh
```

**Windows:** doble clic en `start.bat`

**Manual:**
```bash
pip install flask requests
python server.py TU_CLAVE_AQUI
```

El navegador se abre automáticamente en **http://localhost:5000**

## 🗂 Estructura del proyecto

```
agro-dashboard/
├── server.py          # Servidor proxy Flask (resuelve CORS + caché)
├── start.sh           # Arranque macOS/Linux
├── start.bat          # Arranque Windows
├── README.md          # Este archivo
├── static/
│   └── index.html     # Dashboard completo
└── .cache/            # Caché automático (creado al primer uso)
```

---

## ⚙️ Caché inteligente

El servidor guarda respuestas en `.cache/` para no agotar el límite de 25 calls/día:

| Datos | Caché |
|-------|-------|
| Precios commodities | 4 horas |
| Clima actual | 10 minutos |
| Historial climático | 6 horas |

Para forzar una actualización, borra los archivos en `.cache/`.

## 🔧 Notas técnicas

- **Python 3.8+** requerido
- **Dependencias**: `flask`, `requests` (se instalan automáticamente)
- El servidor corre en `localhost:5000` solamente (no expuesto en red)
- Los datos de Alpha Vantage comienzan desde el año 2000 para la mayoría de commodities
- Las sesiones de futuros siguen calendario NYSE/CME (excluye fines de semana y festivos)
